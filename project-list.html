Featured Project

AI-Powered Inventory System

A modern inventory management system built with PHP, MySQL, and Bootstrap featuring AI-powered forecasting, profit tracking, and a smart chatbot assistant.

Web Application 2026

Overview

What started as a simple CRUD inventory project evolved into a comprehensive system with AI forecasting and chatbot capabilities. Built with PHP and MySQL, this system demonstrates modern web development practices with a focus on user experience and functionality.

Key Features

  • Product & inventory management with real-time tracking
  • Profit tracking with automated margin calculations
  • AI-based stock forecasting for demand prediction
  • Reservation system with unique booking codes
  • Comprehensive analytics and reporting dashboard
  • AI chatbot for natural language inventory queries
  • Glassmorphism UI with fully responsive design

Chatbot Commands

The integrated AI chatbot supports natural language queries for inventory management:

  • "check stock laptop" - View current laptop inventory
  • "reserve 5 123" - Reserve 5 units of product ID 123
  • "show low stock products" - Display items below threshold
  • "top selling products" - View best performing items
  • "today's sales report" - Generate daily sales summary

Technical Implementation

The system uses PHP for server-side logic with MySQL for data persistence. The frontend combines Bootstrap for responsive layouts with custom CSS for the glassmorphism design. JavaScript handles real-time updates and chatbot interactions.

Setup Instructions

Clone the repository and move to your htdocs folder. The system includes a default admin login (admin/admin123) and sample data for testing. The AI features require proper API configuration for full functionality.
